# AgriNova | Smart Farming Hub 

AgriNova is a responsive, visually engaging website designed to showcase smart agriculture solutions in Ethiopia. It blends AI, blockchain, and IoT technologies to empower farmers and connect communities through innovation.

# Features

-  Hero banner with landscape imagery and call-to-action
-  AI Crop Diagnostics, Smart Irrigation, 🔗 Blockchain Traceability
- Testimonial slider with real farmer feedback
- Embedded Google Map for AgriNova HQ (CMC, Addis Ababa)
- Contact form with validation and success alert
- WhatsApp floating button for instant communication
- Custom styling with modern green-themed UI

# Built With

- HTML5
- CSS3 (custom + Bootstrap 5)
- Bootstrap Icons
- Google Maps Embed
